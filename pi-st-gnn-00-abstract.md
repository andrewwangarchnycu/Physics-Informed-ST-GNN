# Building Morphology and Urban Open Space Optimization: A Physics-Informed Graph Neural Network Approach

## Abstract

Rapid urbanization and rising thermal stress have increased the need for performance-driven urban design workflows that can evaluate outdoor thermal comfort efficiently. This study presents a Physics-Informed Spatio-Temporal Graph Neural Network (PIN-ST-GNN) framework for surrogate prediction of the Universal Thermal Climate Index (UTCI) under varying building morphology and urban open-space configurations. In the implemented pipeline, regulation-aware parametric scenarios are generated for an 80 m x 80 m site, using floor area ratio, building coverage ratio, setback, floor-count, and tree-placement constraints. The training dataset is built from EPW-based summer boundary conditions, high-resolution thermal fields, and real-weather calibration using CWB and MOENV IoT sensing data. Urban form is represented as a heterogeneous graph composed of building nodes and air nodes, with semantic object relations and KNN-based spatial contiguity among open-space sensing points. The model combines multilayer relational graph convolution, global climate-time encoding, and temporal LSTM prediction, while physics-informed penalties enforce shading consistency, wind-obstruction regularization, and temporal smoothness. The trained model is deployed through a FastAPI and Grasshopper workflow for near-real-time UTCI prediction and NSGA-II-based search. The implementation demonstrates that thermal comfort can be integrated directly into morphology exploration and multi-objective urban open-space optimization.
